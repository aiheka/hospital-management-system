# Hospital Management System

A web-based Hospital Management System built using Flask and SQLite.

## Features

- Add Patients
- View Patients
- Edit Patients
- Delete Patients
- Add Doctors
- View Doctors
- Book Appointments
- View Appointments

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- HTML

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
